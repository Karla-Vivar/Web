Unidades de medida estaticas y relativas

Las unidades de medida estaticas tal como su nombre indica, se refieren a un metodo para medir
el tamaño de distintos elementos de nuestra pagina como lo serian titulos e imagenes usando de base
pixeles, pulgadas y centimetros, mas sin embargo estas no cambian ni se adaptan cuando son vistas en
dispositivos ya sea mas grandes o pequeños de ahi el nombre de estaticas.

Por otro lado las unidades relativas al igual que su contraparte estatica son unidades de medida con
la diferencia que estas se miden utilizando porcentajes ademas de adaptarse al tamaño del dispositivo y/o
ventana.

flexbox

un flexbox se refiere a un tipo de diseño comunmente usado en html para acomodar datos en hileras y
filas, lo que permite que se adapten de mejor manera a distintos tipos de pantallas, se usa mediante el display : flex en
el elemento "padre".
